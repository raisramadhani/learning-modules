Golden Rules: Bahasa Inggris Profesional

Panduan Praktis Bahasa Inggris Profesional

Lupakan teori rumit. Kuasai pola kalimat ini untuk komunikasi kerja sehari-hari.

Bagian 1: Fondasi Komunikasi

Kuasai ini untuk menyelesaikan tugas dasar dan memastikan tidak ada miskomunikasi.

Meminta dengan Sopan

Pola Kunci: Could you please...?

Contoh: Could you please send the file?

Hindari: Send the file.

Memberi Update Jelas

Pola Kunci: I have + [finished/sent/checked]...

Contoh: I have sent the invoice.

Menyatakan Rencana Berikutnya

Pola Kunci: I will + [kata kerja dasar]...

Contoh: I will call the client tomorrow morning.

Bagian 2: Berkontribusi Aktif dalam Diskusi

Gunakan ini untuk memberi opini, tidak setuju, dan berpartisipasi aktif dalam rapat.

Memberi Opini Terstruktur

Pola Kunci: I think... because...

Contoh: I think we should use this design because it's more user-friendly.

Tidak Setuju Secara Halus

Pola Kunci: I see your point, but...

Contoh: I see your point about the budget, but I'm concerned about the timeline.

Memastikan Pemahaman

Pola Kunci: So, if I understand correctly...

Contoh: So, if I understand correctly, the main priority is quality?

Bagian 3: Komunikasi Tingkat Lanjut

Kuasai ini untuk memengaruhi keputusan dan menangani situasi yang lebih kompleks.

Membingkai Saran

Pola Kunci: By doing X, we can achieve Y

Fokus: Keuntungan bersama

Contoh: By using this software, we can save about 5 hours per week.

Teknik Negosiasi

Pola Kunci: If we were to..., would it be possible to...?

Teknik: Gunakan pengandaian

Contoh: If we were to sign a longer contract, would it be possible to get a better price?
