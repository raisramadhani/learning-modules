📚 Belajar Case Style

Macam Gaya Penulisan Case dalam Code

Pelajari berbagai gaya penulisan yang digunakan dalam pemrograman

🐪

camelCase

Huruf pertama kecil, kata berikutnya kapital

🏛️

PascalCase

Setiap kata dimulai dengan huruf kapital

🐍

snake_case

Semua huruf kecil, dipisah underscore

🍢

kebab-case

Semua huruf kecil, dipisah dash

📢

SCREAMING_SNAKE

Semua huruf kapital, dipisah underscore

📏

flatcase

Semua huruf kecil tanpa pemisah

📋 Panduan Penggunaan

JavaScript/TypeScript

  • • Variables & Functions: camelCase
  • • Classes & Components: PascalCase
  • • Constants: SCREAMING_SNAKE

Python

  • • Variables & Functions: snake_case
  • • Classes: PascalCase
  • • Constants: SCREAMING_SNAKE

CSS/HTML

  • • CSS Classes: kebab-case
  • • HTML Attributes: kebab-case
  • • IDs: kebab-case

Database

  • • Table Names: snake_case
  • • Column Names: snake_case
  • • Package Names: flatcase
