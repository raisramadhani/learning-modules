Dasar-Dasar CSS

Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format dokumen HTML. CSS memungkinkan kita untuk memisahkan konten dari presentasi, membuat website lebih fleksibel dan mudah dikelola.

Keuntungan CSS

  • • Pemisahan konten dan tampilan
  • • Konsistensi desain di seluruh website
  • • Kemudahan maintenance
  • • Loading yang lebih cepat
  • • Responsive design

Contoh CSS Tradisional:

.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

.button {
    background-color: #3b82f6;
    color: white;
    padding: 12px 24px;
    border-radius: 8px;
    border: none;
    cursor: pointer;
}

Pengenalan Tailwind CSS

Apa itu Tailwind CSS?

Tailwind CSS adalah utility-first CSS framework yang menyediakan kelas-kelas siap pakai untuk membangun desain custom dengan cepat. Berbeda dengan framework tradisional, Tailwind memberikan building blocks tingkat rendah untuk membuat desain unik.

Mengapa Tailwind?

  • • Utility-first approach
  • • Highly customizable
  • • Responsive design built-in
  • • Smaller CSS bundle
  • • Consistent design system

Perbandingan: CSS vs Tailwind

CSS Tradisional:
<button class="custom-button">
  Klik Saya
</button>

.custom-button {
  background: blue;
  padding: 12px 24px;
  border-radius: 8px;
}
Tailwind CSS:
<button class="bg-blue-500 px-6 py-3 rounded-lg">
  Klik Saya
</button>

Komponen Tailwind CSS

Spacing (Margin & Padding)

Penjelasan Detail Spacing

Margin Top (mt-)

Margin top adalah ruang kosong di bagian atas elemen yang memisahkan elemen tersebut dari elemen di atasnya. Margin top berguna untuk memberikan jarak vertikal antar elemen.

mt-4 = margin-top: 1rem (16px)
Margin Bottom (mb-)

Margin bottom adalah ruang kosong di bagian bawah elemen yang memisahkan elemen tersebut dari elemen di bawahnya. Sangat berguna untuk memberikan jarak antar paragraf atau section.

mb-6 = margin-bottom: 1.5rem (24px)

Margin Classes

m-0 margin: 0

Menghilangkan semua margin

mt-4 margin-top: 1rem (16px)

Memberikan jarak atas sebesar 16px

mb-8 margin-bottom: 2rem (32px)

Memberikan jarak bawah sebesar 32px

ml-6 margin-left: 1.5rem (24px)

Memberikan jarak kiri sebesar 24px

mr-3 margin-right: 0.75rem (12px)

Memberikan jarak kanan sebesar 12px

mx-auto margin-left & right: auto

Memusatkan elemen secara horizontal

my-12 margin-top & bottom: 3rem (48px)

Memberikan jarak atas dan bawah sebesar 48px

Padding Classes

p-0 padding: 0

Menghilangkan semua padding

pt-2 padding-top: 0.5rem (8px)

Memberikan ruang dalam atas sebesar 8px

pb-6 padding-bottom: 1.5rem (24px)

Memberikan ruang dalam bawah sebesar 24px

pl-4 padding-left: 1rem (16px)

Memberikan ruang dalam kiri sebesar 16px

pr-8 padding-right: 2rem (32px)

Memberikan ruang dalam kanan sebesar 32px

px-5 padding-left & right: 1.25rem (20px)

Memberikan ruang dalam kiri dan kanan sebesar 20px

py-10 padding-top & bottom: 2.5rem (40px)

Memberikan ruang dalam atas dan bawah sebesar 40px
