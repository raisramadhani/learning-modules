Week 13: Java GUI

Memahami pengembangan antarmuka visual pengguna dengan Java

Framework GUI Java

Java menyediakan beberapa framework untuk membuat antarmuka grafis (GUI):

AWT

Abstract Window Toolkit

  • Pustaka awal Java
  • Platform-dependent
  • Heavy-weight components
  • Tampilan mengikuti OS

Swing

Most Popular

  • Platform-independent
  • Light-weight components
  • Tampilan konsisten
  • Komponen lebih kaya

JavaFX

Modern Framework

  • Framework modern
  • CSS styling support
  • 3D graphics
  • Rich media support
Fokus Week Ini: Kita akan fokus pada Swing karena paling umum digunakan dan mudah dipelajari.

Komponen Dasar Swing

import javax.swing.*;

// JFrame - Jendela utama
JFrame frame = new JFrame("My Application");
frame.setSize(400, 300);
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

// JLabel - Menampilkan teks atau gambar
JLabel label = new JLabel("Hello World");

// JButton - Tombol interaktif
JButton button = new JButton("Click Me");

// JTextField - Input teks satu baris
JTextField textField = new JTextField(20);

// JTextArea - Input teks multi-baris
JTextArea textArea = new JTextArea(5, 20);

// JPanel - Container untuk komponen lain
JPanel panel = new JPanel();
panel.add(label);
panel.add(button);

frame.add(panel);
frame.setVisible(true);

Hirarki Komponen Swing:

JFrame (Jendela Utama)
├── JPanel (Container)
    ├── JLabel (Teks/Gambar)
    ├── JButton (Tombol)
    ├── JTextField (Input Teks)
    └── JTextArea (Text Area)

Layout Managers

Layout Manager mengatur posisi dan ukuran komponen secara otomatis.

FlowLayout

Mengatur komponen dalam baris, dari kiri ke kanan.

JPanel panel = new JPanel(new FlowLayout());
panel.add(new JButton("Button 1"));
panel.add(new JButton("Button 2"));
panel.add(new JButton("Button 3"));

BorderLayout

Membagi kontainer menjadi 5 wilayah: North, South, East, West, Center.

JPanel panel = new JPanel(new BorderLayout());
panel.add(new JButton("North"), BorderLayout.NORTH);
panel.add(new JButton("South"), BorderLayout.SOUTH);
panel.add(new JButton("East"), BorderLayout.EAST);
panel.add(new JButton("West"), BorderLayout.WEST);
panel.add(new JButton("Center"), BorderLayout.CENTER);

GridLayout

Membagi kontainer menjadi kisi-kisi sel berukuran sama.

JPanel panel = new JPanel(new GridLayout(3, 2));
panel.add(new JButton("1"));
panel.add(new JButton("2"));
panel.add(new JButton("3"));
panel.add(new JButton("4"));
panel.add(new JButton("5"));
panel.add(new JButton("6"));

Event Handling

Event handling membuat GUI interaktif dengan merespons aksi pengguna seperti klik tombol.

ActionListener untuk Button:

import java.awt.event.*;

JButton button = new JButton("Click Me");

// Method 1: Anonymous Inner Class
button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        System.out.println("Button clicked!");
    }
});

// Method 2: Lambda Expression (Java 8+)
button.addActionListener(e -> {
    System.out.println("Button clicked!");
});

Event Listener Lainnya:

// MouseListener - Deteksi klik mouse
component.addMouseListener(new MouseAdapter() {
    public void mouseClicked(MouseEvent e) {
        System.out.println("Mouse clicked at: " + e.getX() + ", " + e.getY());
    }
});

// KeyListener - Deteksi keyboard
textField.addKeyListener(new KeyAdapter() {
    public void keyPressed(KeyEvent e) {
        System.out.println("Key pressed: " + e.getKeyChar());
    }
});

// WindowListener - Deteksi event window
frame.addWindowListener(new WindowAdapter() {
    public void windowClosing(WindowEvent e) {
        System.out.println("Window closing");
    }
});

Contoh Aplikasi Lengkap

import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class LoginApp extends JFrame {
    private JTextField usernameField;
    private JPasswordField passwordField;
    private JButton loginButton;
    private JLabel statusLabel;
    
    public LoginApp() {
        setTitle("Login Application");
        setSize(350, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLayout(new BorderLayout());
        
        JPanel panel = new JPanel(new GridLayout(3, 2, 10, 10));
        
        panel.add(new JLabel("Username:"));
        usernameField = new JTextField();
        panel.add(usernameField);
        
        panel.add(new JLabel("Password:"));
        passwordField = new JPasswordField();
        panel.add(passwordField);
        
        loginButton = new JButton("Login");
        panel.add(new JLabel());
        panel.add(loginButton);
        
        statusLabel = new JLabel("Please login", JLabel.CENTER);
        
        loginButton.addActionListener(e -> handleLogin());
        
        add(panel, BorderLayout.CENTER);
        add(statusLabel, BorderLayout.SOUTH);
        
        setLocationRelativeTo(null);
        setVisible(true);
    }
    
    private void handleLogin() {
        String username = usernameField.getText();
        String password = new String(passwordField.getPassword());
        
        if (username.equals("admin") && password.equals("1234")) {
            statusLabel.setText("Login successful!");
        } else {
            statusLabel.setText("Invalid credentials!");
        }
    }
    
    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> new LoginApp());
    }
}
