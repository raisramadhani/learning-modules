Week 2: Variables & Data Types

Memahami variabel, tipe data primitif & non-primitif, dan scope variabel di Java

Apa itu Variabel?

Variabel adalah lokasi memori bernama yang menyimpan nilai dan dapat berubah selama eksekusi program.

Bayangkan variabel seperti kotak penyimpanan dengan label yang bisa menyimpan berbagai jenis barang.

Deklarasi Variabel

// Syntax: tipeData namaVariabel = nilai;
int umur = 25;
String nama = "Budi";
double tinggi = 175.5;
boolean sudahMenikah = false;

Scope Variabel

Java memiliki 3 jenis variabel berdasarkan cakupan (scope):

1. Local Variable (Variabel Lokal)

  • Dideklarasikan dalam metode atau blok kode
  • Hanya bisa diakses di dalam metode/blok tersebut
  • Harus diinisialisasi sebelum digunakan

2. Instance Variable

  • Dideklarasikan di dalam class tapi di luar metode
  • Setiap objek punya salinan sendiri
  • Punya nilai default otomatis

3. Static Variable (Class Variable)

  • Dideklarasikan dengan keyword static
  • Milik class, bukan objek individual
  • Hanya ada satu salinan untuk semua objek
public class Mahasiswa {
    // Instance Variable
    String nama;
    int umur;
    
    // Static Variable
    static int jumlahMahasiswa = 0;
    
    public void tampilkanInfo() {
        // Local Variable
        String pesan = "Data Mahasiswa";
        System.out.println(pesan);
        System.out.println("Nama: " + nama);
    }
}
