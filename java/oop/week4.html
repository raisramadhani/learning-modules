Week 4: Inheritance

Memahami pewarisan, method overriding, keyword super, dan jenis-jenis inheritance

Apa itu Inheritance?

Inheritance (Pewarisan) memungkinkan class baru (subclass/child) mewarisi atribut dan metode dari class yang sudah ada (superclass/parent).

Manfaat Inheritance:

  • Code Reusability - Tidak perlu menulis kode yang sama berulang kali
  • Extensibility - Mudah menambah fitur baru tanpa mengubah class yang ada
  • Hierarchical Relation - Menciptakan hubungan hierarki antar class
  • Polymorphism - Memungkinkan polimorfisme (akan dipelajari di Week 6)
// Syntax: class SubClass extends SuperClass
class Animal {
    String name;
    
    void eat() {
        System.out.println(name + " is eating");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println(name + " is barking");
    }
}

// Dog mewarisi atribut 'name' dan method 'eat()' dari Animal

Method Overriding

Method Overriding adalah ketika subclass memberikan implementasi baru untuk metode yang sudah ada di superclass.

Syarat Method Overriding:

  • ‚úì Nama method harus sama
  • ‚úì Parameter harus sama (jumlah dan tipe)
  • ‚úì Return type harus sama atau subtype (covariant)
  • ‚úì Access modifier tidak boleh lebih restriktif

Tidak Bisa Override:

  • ‚ùå Method final
  • ‚ùå Method static
  • ‚ùå Method private
class Animal {
    void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    @Override  // Annotation (opsional tapi recommended)
    void makeSound() {
        System.out.println("Dog barks: Woof! Woof!");
    }
}

class Cat extends Animal {
    @Override
    void makeSound() {
        System.out.println("Cat meows: Meow! Meow!");
    }
}

// Penggunaan
Animal myDog = new Dog();
myDog.makeSound();  // Output: Dog barks: Woof! Woof!

Animal myCat = new Cat();
myCat.makeSound();  // Output: Cat meows: Meow! Meow!

Keyword super

Keyword super digunakan untuk mengakses metode atau konstruktor dari superclass.

Penggunaan super:

  1. Memanggil constructor superclass: super()
  2. Mengakses method superclass: super.methodName()
  3. Mengakses atribut superclass: super.variableName
class Vehicle {
    String brand;
    
    Vehicle(String brand) {
        this.brand = brand;
    }
    
    void displayInfo() {
        System.out.println("Brand: " + brand);
    }
}

class Car extends Vehicle {
    String model;
    
    Car(String brand, String model) {
        super(brand);  // Memanggil constructor parent
        this.model = model;
    }
    
    @Override
    void displayInfo() {
        super.displayInfo();  // Memanggil method parent
        System.out.println("Model: " + model);
    }
}

// Penggunaan
Car myCar = new Car("Toyota", "Avanza");
myCar.displayInfo();
// Output:
// Brand: Toyota
// Model: Avanza
