# Website Pembelajaran OOP Java - Interaktif

Website pembelajaran interaktif untuk memahami konsep Object-Oriented Programming (OOP) di Java dengan visualisasi dan animasi yang menarik.

Website ini terdiri dari 7 halaman pembelajaran:

### Week 1: Class & Object

- Pemahaman dasar OOP
- Konsep Class dan Object
- Constructor
- Memory Allocation (Stack & Heap)
- Garbage Collection
- **Interaksi**: Membuat objek mobil secara visual, demonstrasi memory allocation

### Week 2: Variables & Data Types

- Jenis-jenis variabel
- Tipe data primitif dan non-primitif
- Autoboxing & Unboxing
- Aturan penamaan variabel
- **Interaksi**: Ubah nilai variabel dengan animasi, demonstrasi autoboxing/unboxing

### Week 3: Penguatan & Encapsulation

- Scope variabel (Local, Instance, Static)
- Nilai default variabel
- Prinsip Enkapsulasi
- Access Modifiers
- Getter & Setter
- **Interaksi**: Visualisasi scope variabel, simulasi bank account dengan enkapsulasi

### Week 4: Inheritance

- Konsep pewarisan
- Method Overriding
- Keyword super
- Jenis-jenis inheritance
- **Interaksi**: Inheritance tree visualization, demo method overriding dengan hewan

### Week 6: Polymorphism

- Runtime Polymorphism (Dynamic Binding)
- Compile-time Polymorphism (Static Binding)
- Method Overloading vs Overriding
- **Interaksi**: Demo upcasting dan drawing shapes, calculator overloading

### Week 7: Abstraction

- Konsep abstraksi
- Abstract Class
- Interface
- Multiple Inheritance melalui Interface
- Kapan menggunakan Abstract Class vs Interface
- **Interaksi**: Demo abstraksi dengan mobil, multiple inheritance dengan hewan
