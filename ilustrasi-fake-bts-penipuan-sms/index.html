🛡️ Lindungi Diri Anda

Tips praktis untuk menghindari jebakan penipuan SMS dan melindungi data pribadi Anda

🔍

Cek Kembali

Jangan percaya nama pengirim begitu saja. Penipu dapat menyamar sebagai institusi mana pun.

🚫

Jangan Klik Link Asing

Ketik manual URL resmi atau buka dari aplikasi resmi bank/institusi.

🔐

Jangan Berikan Data Sensitif

Instansi resmi tidak akan meminta password atau OTP melalui SMS.

📞

Verifikasi Langsung

Hubungi call center resmi untuk konfirmasi sebelum mengambil tindakan.

📱

Gunakan Aplikasi Resmi

Selalu gunakan aplikasi mobile banking resmi untuk transaksi keuangan.

🚨

Laporkan

Jika curiga, laporkan ke pihak berwajib atau operator seluler Anda.
